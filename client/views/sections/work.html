Daniboomerang

What you´re seing right now!. With a pure CSS parallax able to simulate 4 levels of profoundity, it pretends to be kind of a CV. An interactive way to show the technologies frameworks and programing languages I know. The freelancer creating and observing the world of connectivity from another perspective, servers and clients connected to the cloud...
I´ve created daniboomerang for several reasons:

  • The main one was to have something where I could clearly show what I´m capable of and how do I do things. You can see the code. You can install the app locally and do whatever pleases you.
  • But I also had a challenge since a long time ago: to do something I´d love to see from the Graphyc Designing and UX point of view. Also create my own images.
  • I wanted something fully responsive. From 200px up to TVs. Something alive that understand that you are using a very small screen and things need to be arranged in a different way.
  • I wanted something completelly tested. With plenty of tests, all of them passed.
  • A Continuous Integration server
  • Achive grate performance: with over 40 SVG images simulating parallax, changing the background position with JS wasn´t possible. Using a pure CSS parallax solution and optimizing images (total < 6MB) was the ony way to reach a smooth scroll
